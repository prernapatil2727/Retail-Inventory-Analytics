# Retail-Inventory-Analytics

## Objective
Analyze retail inventory data to improve stock and sales decisions.

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Google Colab

## Features
- Sales analysis
- Stock analysis
- Supplier analysis
- Data visualization

 ## Key Insights
- Fast-moving products contribute a major share of total sales and should be prioritized for continuous restocking.
- Several low-stock products indicate an immediate need for replenishment to avoid stockouts.
- A small number of categories generate most of the revenue, showing a clear demand concentration.
- Some products have high stock but low sales, indicating slow-moving or dead inventory.
- Certain suppliers dominate inventory supply, which may create dependency risk for the business.
  
## Output
- Charts
- Insights
- Business recommendations
